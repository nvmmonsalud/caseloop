# Future roadmap

## Next: pilot readiness

- Expand InsForge authentication with institution-managed invitations, MFA, and a recurring RLS audit
- Add managed antivirus scanning and OCR to the quarantined PDF/DOCX ingestion workflow
- Pilot the faculty rubric editor and explicit shared-feedback release workflow
- Cohort aggregation with minimum anonymity thresholds
- Evaluation harness for grounding and non-answer-giving behavior

## Then: program workflows

- LMS roster and assignment integrations
- Faculty co-teaching, version history, and reusable case templates
- Longitudinal student reasoning portfolio
- Accessibility audit and internationalization
- Institutional retention, consent, residency, and audit controls

## Later: AI-native LMS

- Cross-course learning objectives and competency maps
- Evidence-based adaptive preparation paths
- Privacy-preserving program benchmarks
- Human-governed assessment support—not autonomous grading

Messaging, calendar, gradebook, and institution administration remain outside the MVP until the core learning loop demonstrates value.
